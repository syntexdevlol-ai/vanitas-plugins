# Vanitas Plugins

Install link (Vanitas Plugins browser):
https://raw.githubusercontent.com/syntexdevlol-ai/vanitas-plugins/main/plugins/vanitas-plugins/

This repo contains multiple plugins under `plugins/`.

## Where To Find It In Discord

After installing and enabling `Vanitas Plugins`:
- Plugins page: open the plugin list, then tap the plugin's wrench/settings button.
- If your client shows a settings row: `Discord Settings -> Keetu/Kettu -> Vanitas Plugins` (may take a few seconds after enabling).

Inside `Vanitas Plugins` you can install/uninstall the other plugins from this repo.
